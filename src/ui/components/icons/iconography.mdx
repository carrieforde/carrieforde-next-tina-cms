## Style API

Use [CSS custom properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) to customize the appearance of icons to suit your theme. Defaults are shown in the code below.

```css
:root {
  --icon-fill: var(--color-default-main, #323f4b);
  --icon-height: var(--icon-height, 1em);
  --icon-stroke: transparent;
  --icon-width: var(--base-font-size, 18px);
}
```
